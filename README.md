A high-performance, immersive cybersecurity portfolio and professional CV designed for modern recruitment. This project showcases technical expertise in ethical hacking, offensive security, and vulnerability research through an interactive "System Breach" simulation.

---

## 🚀 Key Features

### 💻 Interative Cyber Portfolio (`index.html`)
- **System Breach Simulator**: An immersive OSINT & hardware probing simulation with real-time logging and high-fidelity mechanical keyboard SFX.
- **Dynamic Skill Matrix**: Categorized expertise across Offensive Security, Blue Teaming, Cloud Security, and Programming.
- **Impact-Driven Projects**: Detailed showcase of 9+ security tools with direct links to verified repositories.
- **Auto-Print Optimization**: Append `#print` to the URL to instantly trigger a formatted print/PDF dialog.

### 📄 Professional CV (`CV.html`)
- **HR/ATS Optimized**: Clean, professional layout designed to pass through Applicant Tracking Systems.
- **Instant Save**: Automatic print dialog trigger (100ms) for recruiters to save as PDF immediately.
- **Metadata Syncing**: Specialized title tags to ensure the PDF saves as `Kaif_Tarasgar_Cybersecurity_CV.pdf` by default.

---

## 🛠️ Performance & Audio Engine

- **Web Audio API**: Custom-synthesized mechanical "clack" sounds, ensuring 100% reliability regardless of external CDN status or network firewalls.
- **Force Load Architecture**: Assets are pre-buffered to eliminate latency during initial interaction.
